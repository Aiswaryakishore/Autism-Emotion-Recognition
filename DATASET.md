#  Dataset Information

## Overview

The dataset consists of facial images of children with Autism Spectrum Disorder (ASD) used for emotion recognition.

---

## Emotion Classes

- Angry  
- Happy  
- Neutral  
- Sad  

---

## Dataset Size

- Initial Dataset: 881 images  
- After Augmentation: 3524 images  
- Final Balanced Dataset: 6872 images  

---

## Data Processing Steps

- Image preprocessing and cropping  
- Data augmentation to increase dataset size  
- Class balancing to remove bias  
- Train-validation-test split applied  

---

## Data Split

- Training: 4806 images  
- Validation: 1033 images  
- Test: 1033 images  

---

## Note
Dataset is not publicly available due to privacy and ethical considerations.
