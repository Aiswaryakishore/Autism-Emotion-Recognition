# Autism Emotion Recognition System

This project focuses on recognizing emotions in children with Autism Spectrum Disorder using deep learning models.

## Models Used
- CNN
- MobileNet
- ResNet50
- Vision Transformer (ViT)

## Features
- Data augmentation and balancing
- Model comparison
- Grad-CAM visualization
- Accuracy up to 90%

## Technologies
- Python
- TensorFlow / PyTorch
- OpenCV

## How to Run
1. Open the notebook in Google Colab
2. Upload dataset
3. Run all cells
